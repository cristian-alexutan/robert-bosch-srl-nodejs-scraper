# Contributing

Contributiile sunt binevenite! Te rugam sa urmezi acesti pasi:

1. Fork the repository
2. Creaza un branch nou (`git checkout -b feature/ceva`)
3. Fa modificariile si fa commit (`git commit -m 'Adauga ceva'`)
4. Push la branch (`git push origin feature/ceva`)
5. Deschide un Pull Request

### Reguli

- Respecta stilul de cod existent (camelCase, UPPER_SNAKE_CASE pentru constante)
- Testeaza modificarile inainte de a face PR
- Actualizeaza documentatia daca e cazul
- Foloseste mesaje de commit descriptive in engleza sau romana
